# 数据库表设计

数据库选用MongoDB

## 对局

```typescript
{
    // 对局编号
    id: int,
    // 对局所在群组ID
    group_id: int,
    // 对局状态
    state: int,
    // 对局记录
    record: [
        {
            point: int,
            user_id: int
        }, 
        {
            point: int,
            user_id: int
        },
        {
            point: int,
            user_id: int
        }, 
        {
            point: int,
            user_id: int
        }
    ],
    // 对局创建者
    create_user_id: int,
    // 对局创建时间
    create_time: datetime
}
```

索引：

- id
- group_id, record.user_id
- status, create_time
